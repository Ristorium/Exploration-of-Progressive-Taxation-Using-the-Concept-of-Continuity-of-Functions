# Exploration-of-Progressive-Taxation-Using-the-Concept-of-Continuity-of-Functions

A mathematical exploration of progressive taxation using the concept of continuity from Ristorium, visualized with Python and Matplotlib.

해당 파일은 2026년도 1분기의 법령 (소득세법 제55조)을 기준으로 하여 작성되었습니다. 자세한 내용은 "https://www.law.go.kr/법령/소득세법/제55조"를 참고하십시오.
본인 (Ristorium)은 이 파일을 사용하여 세금을 납부하는 과정을 포함하여 생기는 분쟁 등에 대하여 그 어떠한 책임도 지지 않습니다.
